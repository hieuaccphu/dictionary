### Tính năng chính <a name="main-feature"></a>
- Đăng nhập, đăng ký, đăng xuất
- Tìm từ vựng bằng từ điển
- Học từ vựng bằng từ điển
- Lưu trữ từ vựng yêu thích 
- Xóa lưu trữ từ vựng yêu thích 
- Học từ vựng qua các chủ đề Unit
- Lưu trữ chủ đề Unit yêu thích
- Xóa Lưu trữ chủ đề Unit yêu thích
- Luyện kỹ năng Nghe và Đọc qua các bài Test 
- Ôn tập bằng test history 
- Hiển thị, lưu trữ thông tin User 
- Quản lý favorite collection của User
- Quản lý Test history của User
